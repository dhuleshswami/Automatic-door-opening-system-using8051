# Automatic-door-opening-system-using8051
# Automatic Door Lock System using 8051

## Description
This project uses 8051 microcontroller to control a door lock system.

## Components Used
- 8051 Microcontroller
- IR sensor 
- l293d motor driver ic 
- Motor / Solenoid Lock
- 11.0592MHz crystal oscillator
- 33pf capacitor
- 10uf capacitor
- 1k resistor
- push button
- pcb board
  
##software used
-proteus
-keil
-willar

## Working
when person come infront of  door → system checks → door opens → person goes inside or outside  → door closed.
